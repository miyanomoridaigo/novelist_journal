☁ クラウド同期設定（Supabase）

supabase.com で無料プロジェクトを作成し、SQLエディタで下のSQLを実行後、URLとAnon Keyを入力してください。

▶ セットアップ用SQL（クリックで表示） 
create table journal_data (
  key text primary key,
  data jsonb,
  updated_at timestamptz default now()
);
alter table journal_data enable row level security;
create policy "allow all" on journal_data for all using (true);
⚠ このキーはブラウザのlocalStorageに保存されます。共有PCでは使用しないでください。
✅ 設定後は執筆ログ・目標・メモを保存するたびに自動同期されます（3秒後）
